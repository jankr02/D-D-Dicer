## [1.0.0] - 2026-02-01

### Added

- Dice roller with configurable dice groups (count, sides, modifier)
- Preset system for saving and loading dice configurations
- Roll history with session tracking
- Character sheet integration with ability scores and modifiers
- Template library with search and filter functionality
- Copy to clipboard functionality for roll results
- PWA support with install and update notifications
- Dark/Light theme toggle with keyboard shortcut (Ctrl+Shift+D)
- Internationalization support (English and German)
- Responsive design for mobile and desktop

### Technical

- Angular 20 with strict TypeScript configuration
- Service Worker for offline functionality
- LocalStorage persistence for settings, presets, and history
